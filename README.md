# SmartTask Pro

A task management dashboard built with React.

## Features

- Login authentication
- Protected routes
- Task CRUD operations
- Axios API calls
- Fake backend using json-server

## Tech Stack

- React
- React Router
- Axios
- JSON Server

## Run the Project

Install dependencies

npm install

Start frontend

npm run dev

Start fake backend

npx json-server --watch db.json --port 3001
